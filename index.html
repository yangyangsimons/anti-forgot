艾宾浩斯记忆曲线复习表

科学安排复习时间，提高记忆效率

📋 使用说明

一、什么是艾宾浩斯记忆曲线

艾宾浩斯记忆曲线告诉我们，刚学完的内容会快速遗忘，但通过科学的复习时间安排，可以大大提高记忆效果。最佳复习时间点是：学习后20分钟、1小时、1天、2天、4天、7天、15天、30天。

二、如何添加学习内容

  1. 在"输入学习内容"框中填写你要学习的内容（如：英语单词第1组、历史第三章等）
  2. 选择开始学习的日期和时间（默认为当前时间）
  3. 点击"添加到复习计划"按钮
  4. 系统会自动计算出8个复习时间点

建议：每组内容不要太多，单词建议20-30个一组，可以同时管理多个学习内容。

三、复习状态说明

红色 - 未开始：还没有进行这个时间点的复习
橙色 - 进行中：已经开始复习，但还没有完全掌握
绿色 - 已完成：这个时间点的复习已经完成，内容已经牢固掌握

四、日常使用流程

  1. 打开复习表，查看当天需要复习的内容（黄色高亮显示）
  2. 按时间进行复习
  3. 复习后点击状态按钮更新进度：
    • 如果完全记住了 → 点击改为"已完成"（绿色）
    • 如果还有些模糊 → 点击改为"进行中"（橙色）
    • 如果完全忘记了 → 重新学习后改为"进行中"

状态更新原则：一看就知道答案 = 已完成；需要想一下才想起来 = 进行中；完全不记得 = 重新学习后标记进行中

五、重要提醒

  • 时间管理：严格按照表格显示的时间进行复习，过期的复习会标红提醒
  • 灵活调整：状态可以随时修改，如果发现"已完成"的内容又忘记了，可以改回"进行中"
  • 数据保存：所有数据自动保存，关闭浏览器后重新打开数据仍在

六、成功秘诀

  1. 坚持按时复习 - 不要跳过任何一个复习时间点
  2. 诚实标记状态 - 如实反映掌握程度
  3. 重视"进行中"状态 - 这些内容需要额外关注
  4. 分组学习 - 不要一次性添加太多内容
  5. 持续30天 - 完成全部8轮复习才能真正掌握
